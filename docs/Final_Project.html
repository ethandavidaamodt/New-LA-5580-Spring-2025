Park Access in Iowa

Analyzing 10-Minute Walk Access by Census Tract

Project Overview

This project explores which communities in Iowa live within a 10-minute walk of a public park. It combines spatial park data with ACS population estimates to visualize and evaluate equitable access to green space across census tracts.

Data & Methodology

  • Population data: 2022 ACS 5-Year Census Tract Estimates
  • Park data: Iowa Open Data Portal
  • Tools: R (tidycensus, sf), ArcGIS Pro, Tableau
  • Used spatial buffers to estimate 10-minute walk access to parks by checking which census tracts intersect buffer zones.

Key Findings

  • About XX% of tracts had park access within a 10-minute walk.
  • Access was higher in urbanized regions and significantly lower in rural or peripheral census tracts.
  • Population-weighted analysis revealed some dense tracts lacked access despite proximity to major municipalities.

Conclusion

This project highlights key gaps in park accessibility and provides a model for how spatial data can inform equitable planning decisions at the local and regional level.

Personal Reflection

Working on this project helped me strengthen my spatial analysis skills using R and Tableau, while also deepening my understanding of environmental equity. I learned how to clean and merge spatial datasets, construct buffers, and communicate findings clearly through interactive visualizations.
